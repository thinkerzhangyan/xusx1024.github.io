发表于 2017-05-18 | 分类于 Android System Framework

CaptioningManager是什么

  • 看名字：字幕管理
  • Android开发无障碍指南:

    如果应用程序提供视频播放，必须为聋人和重听用户提供字幕和注释来帮助他们理解。视频播放控件必须明确说明字幕是否可用，并提供简单的方式启用字幕。

设备如何设置

google 官方说明

字幕

您可以为设备开启字幕功能以及为字幕选择所需选项（语言、文字和样式）。

开启字幕功能

打开设备的“设置”应用

点按无障碍，然后点按字幕

点按顶部的开启/关闭开关



更改字幕设置

开启字幕功能后，您可以在设置 > 无障碍 > 字幕中调整以下设置。

  • 语言：选择您想要以何种语言显示字幕
  • 文字大小：选择字幕文字大小
  • 字幕样式：为字幕文字和背景选择样式和颜色

您的字幕设置可能不适用于某些应用。

官方文档

简介

官方文档称之为：隐藏式字幕。4.4版本释出。

现在播放HTTP Live Steam(HLS)视频时，VideoView支持WebVTT字幕跟踪，根据用户在系统设置中定义的隐藏式字幕首选项显示字幕跟踪。

您也可以使用addSubtitleSource()方法为VideoView提供WebVTT字幕跟踪。此方法接受一个InputStream,其携带字幕数据和指定该字幕数据格式的MediaFormat对象，您可以使用createSubtitleFormat()指定该对象。这些字幕也会按照用户的首选项显示在视频上。

如果您不使用VideoView显示您的视频内容，则应尽可能使您的字幕叠加层与用户的隐藏式字幕首选项匹配。新的CaptioningManager API允许您查询用户的隐藏式字幕首选项，包括CaptioningManager.CaptionStyle定义的样式，如字体和颜色。如果用户在视频已开始后调整部分首选项，您应侦听首选项的变化，方法是注册一个CaptioningManager.CaptioningChangeListener实例以在任意首选项发生变化时接收回调，然后根据需要更新您的字幕。
